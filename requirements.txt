# ScanLens web app dependencies (CPU-only, for Streamlit Community Cloud).
#
# The PyTorch CPU wheel index is set as the PRIMARY index so that `torch`
# resolves to the small "+cpu" build (not the multi-GB CUDA wheel from PyPI).
# Everything else falls back to PyPI via --extra-index-url.
#
# torch is intentionally left unpinned so it picks the CPU wheel that matches
# whatever Python version Streamlit Cloud uses (e.g. 3.12 -> ..., 3.14 -> 2.12.1+cpu).
# numpy is left unpinned for the same reason (numpy 2 is required on Python 3.13+).
--index-url https://download.pytorch.org/whl/cpu
--extra-index-url https://pypi.org/simple
